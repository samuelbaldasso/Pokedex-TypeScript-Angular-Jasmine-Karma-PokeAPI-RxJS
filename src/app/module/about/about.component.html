About

This is a simple Pokemon app made with Angular 14. It uses the PokeAPI to fetch the pokemon´s data.

The app has two main pages: Home and About.


The Home page is the main page of the app. It shows the name of each Pokemon registered.

The About page is this page. It shows a brief description of the app and the technologies used to build it.


The app uses the following technologies:

  • Angular 14
  • PokéAPI

The app is open source and the code is available on my GitHub.
